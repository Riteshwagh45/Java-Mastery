# ☕ Java Basics

> **The foundation of every Java developer's journey.**

This package covers the fundamental concepts of Java that form the base for advanced topics like OOP, Collections, Multithreading, and Spring Boot. All examples are implemented from scratch to build a strong understanding of Core Java.

---

## 📚 Topics Covered

- Variables
- Data Types
- Keywords & Identifiers
- Literals
- Type Casting
- User Input
- Basic Programs

---

## 🎯 Learning Outcome

After completing this package, you will be able to:

- Write basic Java programs confidently.
- Understand Java syntax and program structure.
- Work with variables and data types.
- Build a strong foundation for advanced Java concepts.

---

## 💼 Interview Focus

- JDK vs JRE vs JVM
- Primitive vs Non-Primitive Data Types
- Type Casting
- Variable Scope
- String Basics
