# 🔄 Iterative Control Statements

> **The foundation of automation and repetitive task execution in Java.**

This package covers Java looping statements used to execute a block of code repeatedly based on a condition. Loops help write efficient, reusable, and optimized programs by eliminating unnecessary code repetition.

---

## 📚 Topics Covered

- while Loop
- do-while Loop
- for Loop
- Nested Loops
- Enhanced for Loop (for-each)
- break Statement
- continue Statement

---

## 🎯 Learning Outcome

After completing this package, you will be able to:

- Solve repetitive problems efficiently.
- Choose the appropriate loop for different scenarios.
- Control loop execution using `break` and `continue`.
- Build logic for patterns, arrays, searching, and algorithmic problems.

---

## 💼 Interview Focus

- Difference between `for`, `while`, and `do-while`
- When to use each loop
- Infinite Loops
- Nested Loops
- `break` vs `continue`
- Enhanced for Loop (for-each)
