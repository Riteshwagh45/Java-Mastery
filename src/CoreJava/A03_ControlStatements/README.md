# 🎯 Control Statements

> **The decision-making mechanism that controls the flow of a Java program.**

This package covers Java control statements that allow a program to make decisions based on conditions. These statements help execute different blocks of code depending on user input or logical expressions.

---

## 📚 Topics Covered

- if Statement
- if-else Statement
- Nested if
- else-if Ladder
- switch Statement
- Enhanced switch (if covered)

---

## 🎯 Learning Outcome

After completing this package, you will be able to:

- Make decisions based on conditions.
- Choose the appropriate control statement for different scenarios.
- Write cleaner and more efficient decision-making logic.
- Build the foundation for loops and real-world applications.

---

## 💼 Interview Focus

- Difference between `if-else` and `switch`
- When to use `switch` instead of `if-else`
- Nested if vs else-if Ladder
- Enhanced Switch Expression (Java 14+)
- Fall-through behavior in switch
