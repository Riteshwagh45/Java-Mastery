# ⚡ Operators

> **The building blocks of every Java expression and calculation.**

This package covers Java operators used to perform arithmetic calculations, comparisons, logical operations, bitwise manipulation, and conditional evaluations. Understanding operators is essential for writing efficient programs and solving logical problems.

---

## 📚 Topics Covered

- Arithmetic Operators
- Unary Operators
- Relational Operators
- Logical Operators
- Assignment Operators
- Ternary Operator
- Bitwise Operators
- Shift Operators

---

## 🎯 Learning Outcome

After completing this package, you will be able to:

- Perform mathematical and logical operations.
- Compare values and evaluate conditions.
- Write concise conditional expressions.
- Build the foundation for control statements and loops.

## 💼 Interview Focus

- Short-Circuit Operators (`&&` vs `&`, `||` vs `|`)
- Increment & Decrement (`++i` vs `i++`)
- Ternary Operator
- Bitwise Operators
